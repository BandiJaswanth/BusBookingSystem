# 🚌 Bus Booking System (Java Swing GUI Application)

A graphical desktop-based Bus Booking System developed using Java Swing. This application provides a user-friendly interface for registering users, logging in, selecting buses, and booking seats visually.

---

## 📌 Features

- User Registration and Login system
- Interactive graphical interface using Swing
- Multiple bus routes and schedules
- Seat selection with visual seat map
- Seat locking to avoid duplicate booking
- Booking confirmation with payment options (simulation)
- Cancel booking feature
- User-specific booking history
- Logout functionality

---

## 🛠️ Technologies Used

- Java
- Swing (GUI)
- AWT (Event Handling & UI)
- Collections Framework (HashMap, ArrayList, HashSet)

---

## ▶️ How to Run

1. Open project in Eclipse / IntelliJ
2. Run `SwiingsBus.java`
3. Register a new user
4. Login with credentials
5. Select bus, choose seat, and book ticket
